# loading_ui_flutter showcase

Interactive showcase app for the [loading_ui](https://github.com/anfeMurillo/Loading-UI-Flutter-Package) Flutter package.

**100 loading widgets** — 36 classic loaders ported from [loading-ui.com](https://loading-ui.com) + 64 dot-matrix loaders ported from [dotmatrix](https://github.com/zzzzshawn/matrix).

## Running

```sh
cd example
flutter run
```

## Features

- Grid of all 100 widgets with live preview
- Size slider (16–96 px)
- Speed multiplier (0.25×–4×)
- 8 colour swatches
- Category filter: All, Spinners, Pulses, Dots, Bars, Text, Special, Matrix
